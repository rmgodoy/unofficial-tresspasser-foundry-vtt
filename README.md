# Trespasser for Foundry VTT

An unofficial Foundry VTT system implementation for the **Trespasser** TTRPG. 

This project aims to provide a robust and automated experience for playing Trespasser on Foundry VTT, focusing on its unique tactical combat and procedural gameplay.

## Features

### Phased Tactical Combat
- **Custom Combat Tracker:** Implements Trespasser's unique phased initiative (Early, Enemy, Late, Critical, End phases).
- **Action Point Management:** Tracking of Action Points (AP) and Focus per combatant.
- **Automated Phase Progression:** Easy navigation through the round's phases.

### Dynamic Interaction System
- **Effects & States:** Robust system for applying and tracking effects and states, including intensity scaling and counters (e.g., Strong vs. Weak).
- **Triggered Effects:** Automated triggers for effects on events like moving, receiving damage, dealing damage, or end of combat.
- **Interaction Cards:** Chat cards with actionable buttons to apply damage, healing, or effects directly to targeted or controlled tokens.

### Comprehensive Item & Actor Support
- **Character & Creature Sheets:** Tailored sheets for player characters (with Callings and Crafts) and creatures.
- **Weapon & Armor Integration:** Built-in systems for active armor reduction and weapon usage.
- **Inventory Management:** Specialized items for Rations, Accessories, Talents, Features, and more.

### User Interface & Experience
- **Responsive Sheets:** Clean and modern UI for all document types.
- **Automatic Styling:** Custom chat card styling and border coloring based on actor/owner colors.
- **V13 Ready:** Built following modern Foundry VTT standards for compatibility with V13.
